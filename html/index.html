AS65342 PKI service

Introduction

You have reached the AS65342 PKI service. From this page, you can learn more about this service, and download the various certificate bundles that are available through this service. You will need to have these certificate bundles installed if you want to use any of the AS65342 services which have TLS enabled.

PKI Structure

The AS65342 PKI is build out of a 3 layered PKI. The as65342-root CA is used as the basis of this PKI, and it's key is stored offline. Below the as65342-root CA is the as65342-intermediary CA, which is used to sign the various production CA's in place inside the AS65342 network. Currently, only one production CA is in place, the as65342-autosign CA.
